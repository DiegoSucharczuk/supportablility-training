## PRODUCTION NOTES

### Color Palette
- Primary: Blue (#2563EB) and Purple (#7C3AED) gradients
- Accents: Teal (#14B8A6), Green (#10B981)
- Text: Gray (#1F2937) on white backgrounds
- Highlights: Yellow (#FCD34D) for important elements

### Typography
- Headlines: Bold, modern sans-serif
- Body: Clean, readable sans-serif
- Code/Technical: Monospace font

### Animation Style
- Smooth, professional transitions (0.3-0.5s)
- Subtle zoom effects on page navigation
- Gentle fade-ins for new content
- Highlight animations for key features
- No jarring or distracting effects

### Music Suggestions
- Corporate, upbeat, motivational
- Think: "Technology", "Innovation", "Success"
- Volume: 20-30% (background only, never overpowering)
- Examples: "Inspiring Corporate" or "Tech Innovation" from royalty-free libraries

### Voiceover Style
- Professional but friendly
- Moderate pace (not too fast)
- Clear enunciation
- Enthusiastic but not overly excited
- Gender: Neutral professional voice (male or female)

### Screen Recording Tips
1. Use 1920x1080 resolution
2. Hide browser bookmarks bar
3. Use incognito/private mode (clean browser)
4. Cursor highlight plugin for visibility
5. Smooth, deliberate mouse movements
6. Rehearse navigation before recording

### Technical Specifications
- **Resolution:** 1920x1080 (Full HD)
- **Frame Rate:** 30 fps or 60 fps
- **Format:** MP4 (H.264)
- **Aspect Ratio:** 16:9
- **Total Duration:** 3:00 minutes
- **File Size Target:** < 50 MB (YouTube optimized)
